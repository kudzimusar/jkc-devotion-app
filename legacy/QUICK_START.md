# 🚀 Quick Start Guide - Enable GitHub Pages

## ⚡ 5-Minute Setup

Follow these simple steps to get your JKC Devotion App live on GitHub Pages!

---

## Step 1: Enable GitHub Pages (2 minutes)

1. **Open your repository settings**
   - Go to: https://github.com/kudzimusar/jkc-devotion-app/settings/pages

2. **Configure deployment**
   - Under "Build and deployment", click **"Configure"**
   - Set **Source**: `Deploy from a branch`
   - Set **Branch**: `main` → `/ (root)`
   - Click **Save**

3. **Wait for deployment**
   - GitHub will automatically deploy your site
   - Takes 1-2 minutes

---

## Step 2: Verify Actions Permissions (1 minute)

1. **Go to Actions settings**
   - Visit: https://github.com/kudzimusar/jkc-devotion-app/settings/actions

2. **Update workflow permissions**
   - Find "Workflow permissions" section
   - Select: **"Read and write permissions"**
   - Check: **"Allow GitHub Actions to create and approve pull requests"**
   - Click **Save**

---

## Step 3: Access Your Live Site! (2 minutes)

Once deployment completes, your app will be live at:

🌐 **https://kudzimusar.github.io/jkc-devotion-app/**

### Test the App:
1. Open the URL in your browser
2. Try clicking on devotion days
3. Test the dark/light mode toggle
4. Mark a day as complete
5. Add a note to a devotion

---

## 🎉 That's It!

Your app is now live and will automatically update whenever you push changes to the main branch!

---

## 📱 Install as PWA (Mobile)

1. Open the site on your phone (Chrome/Safari)
2. Tap the browser menu (three dots)
3. Select "Add to Home Screen" or "Install App"
4. The app will work offline!

---

## 🔄 Future Updates

To update the app:
1. Make changes to files
2. Commit and push to main branch
3. GitHub automatically deploys in 1-2 minutes
4. Changes are live!
