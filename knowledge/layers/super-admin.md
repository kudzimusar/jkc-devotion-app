# Super Admin Console — Church OS Assistant Knowledge

## Console Sections
- Dashboard: Platform MRR, total orgs, active subscriptions
- Organizations: Full list of all churches on Church OS
- AI Ops: Platform-wide AI usage, insight generation stats
- Analytics: Cross-org aggregated metrics (anonymised)
- Churn Risk: Churches showing low activity

## Key Metrics
- MRR: Monthly Recurring Revenue from all subscriptions
- Churn Risk: Orgs with >14 days inactivity flagged as medium,
  >30 days as high, no activity ever as critical
- Paying Orgs: Count of orgs with active subscriptions

## Common Admin Actions
- Invite new church: Use invitation system
- View org details: Click org in Organizations list
- Check AI usage: AI Ops tab → usage by org
- Manage billing: Each org's subscription shown with plan
